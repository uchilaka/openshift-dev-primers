First things first - Source Control

We've setup a source control app for all of your git needs during our event. To start using our very own gogs app, follow these 2 steps:

  • Visit http://gogs-demo.40.76.6.47.nip.io/ to sign up for a new account
  • Complete the signup form be sure to use your Alliance Data email address.
That's it! You'll be all set to start using source control (git).
Ensure that when you commit your code, you use the HTTPS URL for your project NOT SSH. The GOGS environment we've setup does not allow connection via the SSH protocol.
If you need a Google account for your project work, also ensure that you use a Google account that maps to your Alliance Data inbox. Follow this link to setup a Google Account (sans GMail). At the link, be sure to select the Use my current email address instead option.
